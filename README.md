# innovative TodoList 📝 [CE]*
### *Console Edition
## setup 🧱
```bash
cd todo-list-ce

yarn setup
# or
npm setup
```
## build 🛠
```bash
yarn build
```
## start 🚀
```bash
yarn start
```
